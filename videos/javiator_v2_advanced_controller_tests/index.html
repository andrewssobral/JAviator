JAviator V2 Advanced-Controller Tests

A Milestone for the JAviator Project

The two videos below report on our recent flight tests conducted with automatic attitude control, automatic altitude control, and manual position commands. These flight tests represent an important milestone for the JAviator project, because in addition to the helicopter hardware itself, the complete flight control software used in these tests has been designed and implemented by ourselves. Concerning the controller design, we received help and feedback from Gabe Hoffmann (PARC and STARMAC team at the Stanford University), which was of invaluable worth for the pace of the JAviator project.
