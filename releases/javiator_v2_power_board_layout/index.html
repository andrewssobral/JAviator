JAviator V2 Power Board Layout

The four pictures below depict the Printed Circuit Board (PCB) layouts and resulting, fully populated PCBs, in particular, the Power Board (left) and the Sender Board (right), which were designed for the JAviator V2 quadrotor. The layout package includes EAGLE Schematic (SCH) and EAGLE Board (BRD) files for both the Power Board and the Sender Board.
